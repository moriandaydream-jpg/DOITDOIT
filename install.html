기본 설정

1. Supabase SQL

Supabase SQL Editor에서 이 SQL을 실행하면 게시판, 댓글, 관리자 설정, RLS 정책이 만들어져.

2. config.js

배포할 때 이 내용으로 `config.js`를 교체하면 방문자가 같은 설정을 써.

3. 운영 전환

정리 스크립트
  • 공개키를 숨기려면 Cloudflare Worker 프록시를 쓰고 `config.js`에는 `apiBaseUrl`만 넣어.
  • Supabase publishable key를 `config.js`에 넣는 직접 연결 모드는 브라우저에 키가 보이는 구조야.
  • SQL 실행 후 `공유 설정 저장`을 누르면 Supabase에 관리자 ID와 기본 설정이 들어가.
  • 설치가 끝나면 `install.html`, `install.js`, `remove-installer.ps1`은 배포본에서 빼.
  • 브라우저 JS는 서버 파일을 직접 삭제할 수 없어서 정리 스크립트를 제공해.
  • 배경은 관리자 화면이나 `config.js`의 `backgroundUrl`로 나중에 바꿀 수 있어.
