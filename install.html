프로젝트 설정

1. SQL 실행

Supabase SQL Editor에서 아래 내용을 실행하면 게시글 테이블, 공유 설정 테이블, 기본 RLS 정책이 만들어집니다.

2. config.js 생성

배포할 때 이 내용으로 `config.js`를 교체하면 방문자들이 같은 설정을 사용합니다.

3. 권한 조정

  • 게시판 모드는 로그인된 익명 방문자가 글을 쓸 수 있게 되어 있습니다.
  • 블로그 모드는 `forest_site_settings.owner_user_id`와 같은 User ID만 작성하게 되어 있습니다.
  • 더 강하게 막고 싶으면 Supabase에서 RLS 정책을 직접 바꾸면 됩니다.
  • 배경은 `background_url` 또는 `config.js`의 `backgroundUrl`만 바꾸면 교체됩니다.

4. 운영 전환

정리 스크립트

설치가 끝난 뒤에는 `install.html`, `install.js`, `remove-installer.ps1`을 배포본에서 제거하세요. 이 페이지는 서버 파일을 직접 삭제할 수 없어서 정리 스크립트를 함께 제공합니다.
